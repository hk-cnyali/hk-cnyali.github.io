「NOI2018」归程 - 可持久化并查集

Posted at 18-7-18 23:56, Updated at 19-10-16 21:59

Description

给出一个nnn个点mmm条边的无向连通图，每条边有长度和海拔。qqq次询问，每次给出一个水位线ppp ，和一个节点vvv，询问从vvv节点开始，经过一段连续的海拔大于ppp的边后，到1号节点的最短路（前缀海拔大于ppp的边不算贡献）

Constraints

n≤200000n \leq 200000 n≤200000

m,q≤400000m, q \leq 400000m,q≤400000

强制在线

Solution

考场上调了一个上午，最后居然还因为数组开小RE两个点，我还是太弱了

显然对于一个点，答案就是它所在的海拔大于ppp的联通块中disidis_idis​i​​(表示从1号点的最短路)最小的一个

如果不强制在线的话，我们只需要先跑一遍最短路，然后将询问按ppp排序，用并查集维护联通块和答案即可

如果强制在线，则需要使用可持久化并查集

可持久化并查集实际上就是用主席树来维护可持久化数组（fa[]fa[]fa[]和ans[]ans[]ans[]）

然后就能通过此题了
