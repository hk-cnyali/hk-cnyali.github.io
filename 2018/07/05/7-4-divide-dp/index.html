「NOIp模拟赛7.4」divide - Dp

Posted at 18-7-5 07:52, Updated at 19-10-16 21:59

Description

有nnn艘飞船，每艘飞船有一个武力值wiw_iw​i​​。你需要把它们分成两队，每队飞船数目任意。

如果两艘飞船iii和jjj 的武力值相加不小于mmm且不在同一队，那么这两艘飞船就能配合默契

问最多能有多少飞船配合默契（不一定是一一配对，可以多对多），同时还需算出有多少种分队方案可以达到此效果

第二问对109+710^9+710​9​​+7取模

Hint

n≤2000,m≤2∗106,wi≤106n\leq 2000, m\leq 2 * 10^6, w_i \leq 10^6n≤2000,m≤2∗10​6​​,w​i​​≤10​6​​

Solution

将www排序后，考虑在[l,t][l,t][l,t]这段区间中

若 w1+wn<mw_1+w_n < mw​1​​+w​n​​<m，那么w1w_1w​1​​与区间中所有飞船配合都不默契，递归处理[l+1,r][l+1,r][l+1,r]

若 w1+wn>=mw_1+w_n >= mw​1​​+w​n​​>=m，那么wnw_nw​n​​与区间中所有飞船配合都默契，递归处理[l,r−1][l,r-1][l,r−1]

将这些数字依次取出，就能得到一个新的顺序

在这个新的顺序中，我们不关心每一个具体的www的值，只关心它是否能与其他飞船配合默契

设op[i]op[i]op[i]表示第iii艘飞船是否能与其后面的所有飞船配合默契（要么全部都能，要么全部都不能）

再把数组翻转一下便于dp处理（翻转之后op[i]op[i]op[i]表示第iii艘飞船是否能与其前面的所有飞船配合默契）

设f[i][j]f[i][j]f[i][j]表示，考虑完了前iii艘飞船，其中有jjj艘属于第一队的最多默契数

f[i][j]=max(f[i−1][j−1]+op[i]∗(i−j),f[i−1][j]+op[i]∗j)f[i][j]=max(f[i-1][j-1]+op[i] * (i-j), f[i-1][j]+op[i] * j)f[i][j]=max(f[i−1][j−1]+op[i]∗(i−j),f[i−1][j]+op[i]∗j)

（max前半部分表示选进第一队，后半部分表示选进第二队）

方案数类似转移，直接累加即可
