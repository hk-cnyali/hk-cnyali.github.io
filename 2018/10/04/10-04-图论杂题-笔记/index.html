10-04 图论杂题 笔记

Posted at 18-10-4 20:28, Updated at 19-10-16 21:59

今天讲课的内容还是比较和蔼可亲的,听课体验较好.除了最后一道神仙题(没放上来)之外其他的题都比较可做,不是太难.

「CF1037D」

Description

一个 nnn 个点 mmm 条边的无向连通图从 111 号点开始 bfsbfsbfs,可能得到的 bfsbfsbfs 序有很多,取决于出边的访问顺序。

现在给出一个 111 到 nnn 的排列,判断是否可能是一个 bfsbfsbfs 序。

n,m≤2∗105n, m \le 2 * 10^5n,m≤2∗10​5​​

Solution

一个巧妙的做法:

对于每个节点,令其权值为在给定序列中的位置。

然后从 111 号点开始正常的 bfsbfsbfs,出边的访问顺序按照权值从小到大访问。

最后将得到的 bfsbfsbfs 序与给定序列比较,若完全一致则是合法的。

也就是说假定给定的bfsbfsbfs是合法的,按照这个来bfsbfsbfs再验证

「CF901D」

Description

一个 nnn个点 mmm 条边的无向连通图中每个点都有一个权值

现在要求给每条边定一个权值,满足每个点的权值等于所有相连的边权之和,权值可负。

Solution

这道题和「CF990F」Flow Control - 构造有点像,但是还是有细微差别

先随便构造一棵生成树跑一遍,但这时根节点可能还不满足条件

考虑其他的边,一条非树边会形成一个环。

  • 如果是偶环,那么无论这条非树边怎么变,都不会对根节点产生影响。
  • 如果是奇环,那么如果给这条非树边增加或减少权值,根节点会发生 222 的权值变 化,具体是加还是减由路径长度奇偶性决定

「CF859E」

Description

有 mmm 个人,nnn 张椅子,第 iii 个人只能坐在第 uiu_iu​i​​ 或第viv_iv​i​​ 张椅子上。

求有多少种方案满足没有人坐在同一张椅子上。

Solution

这种题有点套路啊...

把椅子作为点,人作为边,变成一个图,每个连通块可以分开考虑。

假设某个连通块中有 vvv 个点,eee 条边,由于连通,有 v−1≤ev - 1 \le ev−1≤e,并且若 e>ve > ve>v 则无解,所以 eee 只有 v−1v - 1v−1 和 vvv 两种取值。

  • 假如 e=v−1e = v - 1e=v−1,那么该连通块有 vvv 种方案:考虑枚举每个点不放的情况,其他的点都可以唯一确定。
  • 假如 e=ve = ve=v 且环长 >1> 1>1,那么该连通块有 222 种方案:考虑环上的一条边,这条边的放法确定后其他的都可以唯一确定。

「CF852D」

Description

给定一个 vvv 个点 eee 条边的带权无向图,在图上有 nnn 个人,第 iii 个人位于点 xix_ix​i​​ ,一个人通过一条边需要花费这条边的边权的时间。现在每个人可以自由地走。求最短多少时间后满足结束后有人的节点数 ≥m\ge m≥m n,v≤500n, v \le 500n,v≤500

Solution

先floyd floydfloyd 预处理出两两之间的距离。

然后可以二分答案。二分答案之后,每个人向能走到的点连边。

可以发现合法的条件就是最大匹配数 ≥m\ge m≥m,跑二分图匹配就可以了。

「CF845G」

Description

给定一个 nnn 个点 mmm 条边的带权无向连通图。

从 111 号点出发,初始花费是 000,每经过一条边花费就会异或上这条边的权值,当你到达 nnn 号点时可以选择停下来。

求停下来时的最小花费。

n,m≤105n, m \le 10 ^5n,m≤10​5​​

Solution

这道题前半部分还是很巧妙的,只可惜还不会线性基,只是大概了解,也暂时不打算学

假设 sss 和 tt t是两条 111 到 nnn 的路径,CCC 是图中所有环的集合。那么 ttt 的权值一定可以通过 sss 的权值异或上 CCC 中一些环的权值来得到。

那先随便搞一棵生成树,然后对于每条非树边,都形成了一个环,这些环构成的集合是 DDD

那么 CCC 中的每个环的权值都可以通过 FFF 中的一些环的权值异或和来得到。

于是只要求出 DDD 的线性基,然后随便搞一条路径在线性基上跑一下。
