「CF911F」

Description

给定一棵 nnn 个节点的树，你可以进行 n−1n - 1n−1 次操作，每次操作步骤如下：

  • 选择 u,vu, vu,v 两个度数为 111 的节点。
  • 将 u,vu, vu,v 之间的距离加到 ansansans 上。
  • 将 uuu 从树上删除。

求一个操作序列使得 ansansans 最大。

Solution

先把直径拿出来，将直径外的点一个一个的和直径中的某一个端点配对并删掉。最后再将直径删掉。

这个做法显然是对的。。。

「CF1051F」

Description

给定一张nnn 个点 mmm 条边的带权无向联通图，qqq 次询问，每次询问 uiu_iu​i​​ 到 viv_iv​i​​ 的最短路长度。

n,q≤105,m−n≤20n, q \le 10^5 , m - n \le 20n,q≤10​5​​,m−n≤20

Solution

首先随便搞一棵生成树，把这些非树边边的端点标记为特殊点。

对于每一个询问，如果最短路只经过生成树上的边，就可以直接计算。

否则一定经过了一个特殊点，把所有特殊点预处理出最短路，然后枚举这个特殊点，然后更新答案就行了

「CF1042F」

Description

给定一棵 nnn 个节点的树，定义叶子是度数为 111 的节点，点集 SSS 是好的如果满足任意两个 SSS 中的点之间的距离 ≤k\le k≤k

现在要求将所有叶子分成若干不相交的好的集合，请问最少分成多少个好的集合。

n,k≤106n, k \le 10^6n,k≤10​6​​

Solution

先随便选一个不是叶子的根，然后可以自底向上贪心。

设 fpf_pf​p​​ 表示 ppp 为根的子树内已完成的好的集合的数量，gpg_pg​p​​ 表示未完成的集合中离 ppp 最远的叶子的距离。

（实际上这里指的未完成的集合是指集合内距离都≤k\le k≤k，如果大于kkk则必须再新加入一个集合）

考虑如何计算

首先将所有儿子的 fff 先求和，然后将所有儿子的 ggg 排序，不断检查 一下最大的 ggg 和次大的 ggg 相加是否大于 kkk，如果大于，那么就将最大的 ggg 删掉， 并把 fpf_pf​p​​ 加上 1，循环过程。否则就令 gpg_pg​p​​ 等于最大的 ggg 加 111并跳出。

来历不明的题

Description

给定一棵 nnn 个点的带边权树，mmm 次询问，每次询问给出两个值 p,kp, kp,k，求以下值：

∑q∣dis(p,q)≤kdis(q,p)\sum_{q| dis(p,q)\le k} dis(q, p)∑​q∣dis(p,q)≤k​​dis(q,p)

Solution

动态点分治（点分树）裸题

首先点分治，把分治结构建好。

每个分治中心要记录当前分治区域所有点到分治中心的距离，以及分别记录每个分 治子区域中的所有点到分治中心的距离，以及当前分治中心到其分治祖先的距离。

每次查询时在每个分治中心根据以上信息查询就可以了。
