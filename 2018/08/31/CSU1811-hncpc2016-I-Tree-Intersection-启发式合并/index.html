「CSU1811」 Tree Intersection - 启发式合并

Posted at 18-8-31 14:55, Updated at 19-10-16 21:59

题目链接：传送门

Description

给你一颗nnn个点的树 每个点有一种颜色 问你对于每一条边，去掉这条边之后剩下的两个部分的颜色集合交集的大小

Constraints

n≤105n\le 10^5n≤10​5​​

Input

The input contains at most 15 sets. For each set:

The first line contains an integer nnn (2≤n≤1052\le n\le10^52≤n≤10​5​​).

The second line contains nnn integers c1,c2,...,cnc_1,c_2,...,c_nc​1​​,c​2​​,...,c​n​​ (1≤ci≤n1\le c_i\le n1≤c​i​​≤n).

The i-th of the last (n−1)(n-1)(n−1) lines contains2 22 integers ai,bi (1≤ai,bi≤n1\le a_i,b_i\le n1≤a​i​​,b​i​​≤n).

Output

For each set,(n−1) (n-1)(n−1) integers R1,R2,...,Rn−1R_1,R_2,...,R_{n-1}R​1​​,R​2​​,...,R​n−1​​

Solution

首先题意可以转化为 求出每个子树内每种颜色出现次数，如果次数等于000或者等于这种颜色在整棵树里出现的次数的话显然这个颜色就没有贡献，否则就有贡献

所以我们就可以直接树上启发式合并，用map或者线段树合并都行
