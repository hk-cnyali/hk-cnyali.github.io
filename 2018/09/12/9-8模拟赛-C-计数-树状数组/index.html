「NOIp模拟赛9.8」C - 计数 + 树状数组

Posted at 18-9-12 20:41, Updated at 20-2-25 23:18

Description

按照字典序升序给所有[1,n][1, n][1,n] 的排列标号,从 111 开始。 小 DDD 曾经从中选择了一个排列 p1,p2,...,pn{p_1, p_2 ,..., p_n }p​1​​,p​2​​,...,p​n​​,但现在他忘记了这个排列的部分元素,于是 他决定用000 表示他忘记的元素 pip_ip​i​​ 他想知道所有可能被选中的排列的序号之和,对 998244353998244353998244353 取模。

Constraints

n≤3×105n\le 3\times 10^5n≤3×10​5​​

Solution

假设有 xxx 个未确定元素,那么有 x!x!x! 个可行排列 P 。我们令标号从 000 开始,只需给答案加上 x!x!x! 即可,即只需要计算有多少个排列排在它前面。

容易推出一个排列 aaa 的序号:

∑i=1n(n−i)!(ai−1−∑j=1i−1[aj<ai])\sum_{i=1}^{n}{(n-i)!(a_i-1-\sum_{j=1}^{i-1}{[a_j<a_i]})}∑​i=1​n​​(n−i)!(a​i​​−1−∑​j=1​i−1​​[a​j​​<a​i​​])

那么答案为

∑a∈P∑i=1n(n−i)!(ai−1−∑j=1i−1[aj<ai])\sum_{a\in P}\sum_{i=1}^{n}{(n-i)!(a_i-1-\sum_{j=1}^{i-1}{[a_j<a_i]})}∑​a∈P​​∑​i=1​n​​(n−i)!(a​i​​−1−∑​j=1​i−1​​[a​j​​<a​i​​])

=∑i=1n(n−i)!∑a∈P(ai−1−∑j=1i−1[aj<ai]) = \sum_{i=1}^{n}{(n-i)! \sum_{a\in P}(a_i-1-\sum_{j=1}^{i-1}{[a_j<a_i]})}=∑​i=1​n​​(n−i)!∑​a∈P​​(a​i​​−1−∑​j=1​i−1​​[a​j​​<a​i​​])

=∑i=1n(n−i)!(∑a∈P(ai−1)−∑j∈[1,i),pj=0∑a∈P[aj<ai]−∑j∈[1,i),pj≠0∑a∈P[aj<ai]) = \sum_{i=1}^{n}{(n-i)!( \sum_{a\in P}(a_i-1)-\sum_{j\in [1,i), p_j=0} \sum_{a\in P} {[a_j<a_i]}} - \sum_{j\in [1,i), p_j\neq0} \sum_{a\in P} {[a_j < a_i]})=∑​i=1​n​​(n−i)!(∑​a∈P​​(a​i​​−1)−∑​j∈[1,i),p​j​​=0​​∑​a∈P​​[a​j​​<a​i​​]−∑​j∈[1,i),p​j​​≠0​​∑​a∈P​​[a​j​​<a​i​​])

∑a∈P(ai−1)=(pi−1)∗x!\sum_{a\in P}(a_i - 1) = (p_i - 1) * x!∑​a∈P​​(a​i​​−1)=(p​i​​−1)∗x!

∑j∈[1,i),pj=0∑a∈P[aj<ai]=(∑j∈[1,i)[pj=0])∗(∑k∈S[k<pi])∗(x−1)!\sum_{j\in [1,i), p_j=0} \sum_{a\in P} {[a_j < a_i]} = (\sum_{j\in[1, i)}[p_j=0]) * (\sum_{k\in S}[k<p_i]) * (x - 1)!∑​j∈[1,i),p​j​​=0​​∑​a∈P​​[a​j​​<a​i​​]=(∑​j∈[1,i)​​[p​j​​=0])∗(∑​k∈S​​[k<p​i​​])∗(x−1)!
