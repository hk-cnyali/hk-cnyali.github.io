Description

给你一个长度为NNN的序列a1,a2,...,aNa_1, a_2,...,a_Na​1​​,a​2​​,...,a​N​​，你可以进行如下操作：

选择一个正整数kkk，对于序列aaa中每一个元素可以选择对kkk取模也可以什么都不干

无论你选择多少个元素对kkk取模，一次操作都要花费2k2^k2​k​​的代价

问是否能将序列aaa变成序列bbb，如果可以则输出最小代价

Solution

那么显然我们就能从大到小枚举kkk，如果只用k−1k-1k−1的所有数能满足题意的话kkk就对答案没有贡献，否则就有贡献
