CF1039D

Description

有一棵nnn个点的树.

一个简单路径的集合被称为”kkk合法”当且仅当: 树的每个点至多属于其中一条路径(可以不属于), 且每条路径恰好包含kkk个点.

对于k∈[1,n]k \in [1, n]k∈[1,n], 求出”kkk合法”路径集合的最多路径数.

n≤105n \le 10^5n≤10​5​​

Solution

[分块]

对于每个确定的kkk，显然可以直接O(n)O(n)O(n)从下往上贪心，尽量靠下方合并

发现当k>nk > \sqrt nk>√​n​​​时答案≤n\le \sqrt n≤√​n​​​，所以当k∈(n,n]k\in (\sqrt n, n]k∈(√​n​​​,n]时，按答案分块，每块二分右端点

删除一个点之后, 如果以uuu两个儿子为根的子树大小一样, 重儿子保持不变.

「十二省联考 2019」春节十二响

Description

给定一棵nnn个点, 以111为根的树. 每个点有权值wiw_iw​i​​

你需要把所有点划分到若干个集合中, 满足每个集合内不存在两点互为祖先/后代.

每个集合的权值为其内点的权值的最大值. 一种划分的权值为划分出的所有集合的权值之和.

求划分的权值的最小值.

n≤2∗105n\le 2*10^5n≤2∗10​5​​

Solution

[启发式合并] [贪心]

对于任意一条到根的链或其子链, 其上的所有点一定分属不同的集合

贪心，每个结点的子链上面一一向比自己大的配对，即对每个点的任意两条子链，按从大到小的顺序依次合并（最大的和最大的，次大的和次大的大的和kkk的顺次合并）

用个堆维护，启发式合并
