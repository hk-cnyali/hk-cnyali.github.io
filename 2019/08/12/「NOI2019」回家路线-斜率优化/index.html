「NOI2019」回家路线 - 斜率优化

Posted at 19-8-12 20:34, Updated at 19-10-16 21:59

你需要规划一条路线，使得从起点 111 号点出发，沿着这条路线到达终点 nnn 号点。

假设路线依次经过的边为 a1,a2,...,ak{a_1,a_2,...,a_k}a​1​​,a​2​​,...,a​k​​，则需要保证 而这条路线的代价是 qak+∑i=2kf(pai−qai−1)+f(pa1)\displaystyle q_{a_k}+\sum_{i=2}^{k}f(p_{a_i}-q_{a_{i-1}})+f(p_{a_1})q​a​k​​​​+​i=2​∑​k​​f(p​a​i​​​​−q​a​i−1​​​​)+f(p​a​1​​​​)，其中f(x)=Ax2+Bx+Cf(x)= Ax^2 + Bx + Cf(x)=Ax​2​​+Bx+C

你需要使得你规划的路径的代价最小，输出这个最小代价。

Solution

设f[i]f[i]f[i]表示经过第iii条边后的最小代价，那么

f[i]=minyj=xi,qj≤pi{f[j]+A(pi−qj)2+B(pi−qj)+C}=minyj=xi,qj≤pi{f[j]+Aqj2−Bqj−2Apiqj}+Api2+Bpi+C \begin{aligned} f[i]&=\min_{y_j=x_i,q_j\le p_i}\{f[j]+A(p_i-q_j)^2+B(p_i-q_j)+C\}\\ &=\min_{y_j=x_i,q_j\le p_i}\{f[j]+Aq_j^2-Bq_j-2Ap_iq_j\}+Ap_i^2+Bp_i+C \end{aligned} ​f[i]​​​​=​y​j​​=x​i​​,q​j​​≤p​i​​​min​​{f[j]+A(p​i​​−q​j​​)​2​​+B(p​i​​−q​j​​)+C}​=​y​j​​=x​i​​,q​j​​≤p​i​​​min​​{f[j]+Aq​j​2​​−Bq​j​​−2Ap​i​​q​j​​}+Ap​i​2​​+Bp​i​​+C​​

显然可以在每个点内部进行斜率优化，方法见此

设j<kj < kj<k，若jjj比kkk优，那么有

f[j]+A(pi−qj)2+B(pi−qj)<dp[k]+A(pi−qk)2+B(pi−qk)⇒(f[j]+Aqj2−Bqj)−(f[k]+Aqk2−Bqk)qj−qk>2Api \begin{aligned} &f[j] + A(p_i - q_j)^2 + B(p_i - q_j) < dp[k] + A(p_i - q_k)^2 + B(p_i - q_k)\\\\ \Rightarrow &\frac{(f[j] + Aq_j^2 - Bq_j) - (f[k] + Aq_k^2-Bq_k)}{q_j - q_k}>2Ap_i \end{aligned} ​​​⇒​​​f[j]+A(p​i​​−q​j​​)​2​​+B(p​i​​−q​j​​)<dp[k]+A(p​i​​−q​k​​)​2​​+B(p​i​​−q​k​​)​​q​j​​−q​k​​​​(f[j]+Aq​j​2​​−Bq​j​​)−(f[k]+Aq​k​2​​−Bq​k​​)​​>2Ap​i​​​​

因为推出来是<<<，所以要把条件改成kkk比jjj优，即

(f[j]+Aqj2−Bqj)−(f[k]+Aqk2−Bqk)qj−qk<2Api \frac{(f[j] + Aq_j^2 - Bq_j) - (f[k] + Aq_k^2-Bq_k)}{q_j - q_k}<2Ap_i ​q​j​​−q​k​​​​(f[j]+Aq​j​2​​−Bq​j​​)−(f[k]+Aq​k​2​​−Bq​k​​)​​<2Ap​i​​
