BZOJ4289 Tax

Description

给出一个nnn个点mmm条边的无向图，经过一个点的代价是进入和离开这个点的两条边的边权的较大值

求从起点111到点nnn的最小代价。起点的代价是离开起点的边的边权，终点的代价是进入终点的边的边权

n≤105,m≤2×105n\le 10^5, m\le 2\times10^5n≤10​5​​,m≤2×10​5​​

Solution

[最短路]

把一条边拆成两条有向边，再把有向边视为点，然后按题意建图

直接建图边数是O(m2)O(m^2)O(m​2​​)的，考虑优化

对于一个原图中的点而言，设新图中连进它的边集为SSS，从它连出去的边集为TTT

不难发现，这里的SSS和TTT是一一对应的（有向边是由无向边拆得）

将S,TS, TS,T按边权从小到大排序，连边方法：(Si,Ti,wi),(Ti,Ti+1,wi+1−wi),(Ti+1,Ti,0)(S_i, T_i, w_{i}), (T_i, T_{i+1}, w_{i+1} - w_i), (T_{i + 1}, T_{i}, 0)(S​i​​,T​i​​,w​i​​),(T​i​​,T​i+1​​,w​i+1​​−w​i​​),(T​i+1​​,T​i​​,0)

正确性显然，是一个经典套路

CF1137C Museums Tour

Description

一张nnn个点mmm条边的有向图，定义周期为 ddd 天，从第一个周期的第 111 天开始从 111 号点出发，一条边耗时一天。

点在每个周期中会有部分时间有 111 的点权，同一点不能重复获得点权，问能够得到的最大价值。

n,m≤105,d≤50n, m\le10^5, d\le 50n,m≤10​5​​,d≤50

Solution

[强连通分量] [缩点]

注意到ddd很小，于是可以往分层图上想

按天分层，对于原图中x→yx\rightarrow yx→y的边，连(x,k)→(y,k+1)(x, k) \rightarrow (y, k + 1)(x,k)→(y,k+1)的边（注意不用真的连出来）

缩点后求最长路即可

这里缩完点后每个新点的sizesizesize并不是直接把它包含的所有点的权值加起来，因为可能有点重复的情况，要用个vis搞下

为什么每个点只会被算一次贡献？

即若(x,k)(x, k)(x,k)能走到(x,k′)(x, k')(x,k​′​​)，那么它们为什么一定在同一个强连通分量中？

显然，考虑从(x,k)(x, k)(x,k)到(x,k′)(x, k')(x,k​′​​)经过的路径序列，把它再重复d−1d-1d−1次后，一定能回到(x,k)(x, k)(x,k)
