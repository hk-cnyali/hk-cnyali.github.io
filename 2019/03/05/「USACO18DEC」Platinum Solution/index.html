Balance Beam

Links

Luogu P5155

Description

给定一个长为nnn的序列，可以选择以12\frac{1}{2}​2​​1​​的概率进行左右移动，也可以结束并得到当前位置上的收益fif_if​i​​（fif_if​i​​给出）

求从每个位置开始时使用最优策略的最大期望收益是多少

Constraints

n≤105n\leq 10^5n≤10​5​​

Solution

一看到12\frac{1}{2}​2​​1​​的概率随机游走就自然而然地想到了赌徒破产模型

显然，以iii开始的答案，要么是直接在iii结束，要么是往左/右随机游走到一个收益尽可能大的点并停下

假设往左走到xxx点，往右走到yyy点，那么它们对以iii位置开始的答案贡献即为y−iy−xfx+i−xy−xfy\frac{y-i}{y-x}f_x + \frac{i-x}{y-x}f_y​y−x​​y−i​​f​x​​+​y−x​​i−x​​f​y​​，我们就是要找到点对(x,y)(x,y)(x,y)使得这个东西最大

这个贡献的式子很优美，稍微观察一下就会发现它就是在坐标系中(x,fx)(x, f_x)(x,f​x​​)，(y,fy)(y, f_y)(y,f​y​​)这两点的连线与直线x=ix=ix=i的焦点的纵坐标

我们要使得交点纵坐标尽量大，那么显然只需要维护一个上凸壳即可（真的很显然，我连图都懒得放了）

Sort It Out

Links

Luogu P5156

Description

给定一个长为nnn的排列，可以选择一个集合SSS，不断重复比较SSS内的每个元素与原排列相邻元素的大小关系，若相邻元素不递增则交换。

你需要选择一个集合SSS使得操作后排列变为上升序列

求满足这样条件的，且集合大小最小的集合中字典序第kkk小的集合

Constraints

n≤105,k≤1018n\le 10^5, k\le 10^{18}n≤10​5​​,k≤10​18​​

Solution

原题意很绕，首先不难注意到这样一个事实：若操作集合SSS后能够将序列排好序，那么SSS中的每个元素在最后序列中都会被放到该放的位置上去（元素xxx被移动到第xxx位），且集合SSS以外的元素的相对位置关系不变。因此SSS以外的元素就必须是一个单调上升子序列

于是题目就能转化为，求原序列字典序第kkk大的最长上升子序列（把原问题对偶）

对于这个问题，我们只需要在树状数组求LIS的时候多记一个以当前点开头的LIS数量，并用一个vector记录一下每一次的转移信息，最后把vector元素从大到小排序，用类似于线段树求第kkk大的方法计算一下即可

注意在统计第kkk大时，需要保证元素下标单调递增；以及统计次数的时候有可能会爆long long，所以要对101810^{18}10​18​​取min

具体实现不是很难

The Cow Gathering

Links

Luogu P5157

Description

给定一棵nnn个节点的树，有mmm个限制(u,v)(u,v)(u,v)表示uuu必须在vvv前删除

问每次删除一个叶子节点，可能最后一个留下的点的集合

Constraints

n,m≤105n, m\le 10^5n,m≤10​5​​

Solution

这应该是这一场里最傻逼的一道题了

考虑添加一条限制(u,v)(u,v)(u,v)后对答案集合产生的影响

显然，以vvv为根，在uuu的子树中的点都不可能最后留下了

于是我们每次把这些点标记一下即可

标记的话，可以直接对以111为根，一段连续的dfs序操作。具体来说，若以111为根时，vvv在uuu的子树中，则倍增/二分找到vvv在uuu的哪个儿子的子树中，对这棵子树外的部分修改；否则直接对uuu的子树修改

可以用前缀和或者树状数组维护

注意，最后还需要跑一遍拓扑排序特判一下无解的情况
