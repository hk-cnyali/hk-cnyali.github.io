扩展欧几里得算法

Posted at 17-8-21 23:23, Updated at 19-10-16 21:59

基本算法

对于不完全为 0 的非负整数 a，b 必然存在整数对 x，y ，使得 ax + by = gcd(a, b);

证明

  • 当b = 0 时：gcd(a, b) = a 此时易知x = 1,y = 0

  • 当b ≠ 0 时： 设ax0+by0=gcd(a,b)ax_0 + by_0 = \gcd(a,b)ax​0​​+by​0​​=gcd(a,b)

由于gcd(a,b)=gcd(b,a%b)\gcd(a, b) = \gcd(b, a\% b)gcd(a,b)=gcd(b,a%b)

我们如果设bx1+(a%b)y1=gcd(b,a%b)bx_1 + (a \% b)y_1 = \gcd(b, a\%b)bx​1​​+(a%b)y​1​​=gcd(b,a%b)

那么ax0+by0=bx1+(a%b)y1ax_0 + by_0 = bx_1 + (a \% b)y_1ax​0​​+by​0​​=bx​1​​+(a%b)y​1​​

即

展开移项得:ax0+by0=ay1+b(x1−⌊ab⌋y1)ax_0 + by_0 = ay_1 + b(x_1 - \lfloor {\frac{a}{b}}\rfloor y_1)ax​0​​+by​0​​=ay​1​​+b(x​1​​−⌊​b​​a​​⌋y​1​​)

所以我们有： x0=y1 x_0 = y_1 x​0​​=y​1​​ y0=x1−⌊ab⌋y1 y_0 = x_1 - \lfloor {\frac{a}{b}}\rfloor y_1y​0​​=x​1​​−⌊​b​​a​​⌋y​1​​

于是我们在递归求gcd时回溯求x0x_0x​0​​和y0y_0y​0​​即可
